# PRODUCT.md

## Product Context
- **Name:** RanahInsight
- **Tagline:** Platform Analisis Sentimen Ulasan & Perencana Rute Wisata Sumatera Barat Berbasis AI.
- **Register:** `product`
- **Domain:** Pariwisata, Analitik Data NLP, Web Scraping, dan Rute Perjalanan.

## Tech Stack
- **Web Frontend:** Next.js (React 19, Tailwind CSS v4, shadcn/ui)
- **Mobile App:** Flutter (Dart, Riverpod, GoRouter)
- **Backend Service:** NestJS (TypeScript, Prisma ORM, PostgreSQL)
- **ML & NLP Engine:** Python (NLP Topic Modeling, HDBSCAN/BIRCH, Sentiment Lexicon)

## Design System & Rules
- **Color Space:** OKLCH for dynamic palette styling.
- **Theme Support:** Light and Dark mode integrated via `next-themes`.
- **Typographic Scale:** Clamp heading system for clean display sizing, text-wrap balance for headers.
- **Absolute Bans:** No side-stripe borders on card list items, no gradient text combining background-clip, no uppercase kicker eyebrows on every section.
