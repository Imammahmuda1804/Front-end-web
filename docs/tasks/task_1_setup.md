# Task 1: Setup & Core Architecture

Membangun fondasi utama frontend agar siap untuk pengembangan fitur lanjutan.

## Rincian Pekerjaan:
1. Inisialisasi Next.js 15 App Router menggunakan TypeScript dan Tailwind CSS.
2. Inisialisasi dan konfigurasi Shadcn/ui.
3. Setup Theme provider (next-themes) untuk dark/light mode.
4. Setup folder structure sesuai standar (features, components, services, lib, dsb).
5. Setup Zustand store dasar (contoh: AuthStore, ThemeStore).
6. Setup TanStack Query (QueryClientProvider).
7. Konfigurasi Axios Interceptors untuk BaseURL backend, auto-inject Token, dan refresh token handling.
