# Task 6: Compare Destination Page (User)

Menyediakan fitur bagi pengguna untuk membandingkan performa analitik dari dua destinasi secara langsung.

## Rincian Pekerjaan:
1. **Compare Selector Component**: Dropdown/searchable select untuk memilih destinasi pertama dan destinasi kedua.
2. **Radar Comparison Chart**: Visualisasi sentimen dan metrik dasar menggunakan Radar Chart.
3. **Sentiment Comparison Chart**: Bar chart berdampingan untuk perbandingan rasio positif/negatif.
4. **Trend Comparison Chart**: Line chart multi-line untuk melihat tren seiring berjalannya waktu.
5. **Topic Comparison Card**: Ringkasan topik teratas dari masing-masing destinasi untuk perbandingan.

## Endpoint Terkait:
- `GET /api/analytics/compare`
