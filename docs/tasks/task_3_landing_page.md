# Task 3: Landing Page (User)

Membangun halaman depan utama aplikasi untuk menampilkan rekomendasi destinasi, pencarian, dan tren terbaru.

## Rincian Pekerjaan:
1. **Hero Section**: Membuat banner utama dengan desain menarik dan fitur *Semantic Search Bar*.
2. **Trending Carousel**: Menggunakan Embla Carousel untuk menampilkan destinasi terpopuler / peringkat tertinggi.
3. **Topic Filter Chips**: Menampilkan daftar filter topik (misal: "Keluarga", "Alam", "Murah").
4. **Recommendation Grid**: Menampilkan *cards* destinasi rekomendasi dari AI.
5. **Mini Analytics**: Menampilkan statistik singkat sistem di bagian bawah landing page.

## Endpoint Terkait:
- `GET /api/destinations/recommendations`
- `GET /api/destinations/ranking`
- `GET /api/topics`
- `GET /api/analytics/dashboard`
